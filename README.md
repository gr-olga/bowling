# Bowling Score Calculator

## Overview

This project simulates a game of bowling, allowing players to manually input their scores and calculate their total scores based on the traditional rules of bowling. The game consists of ten frames, and in each frame, the player has two chances to knock down as many pins as possible. The project is built using React and TypeScript, providing a visual representation of the game and its scores.

## Features

### Minimum Requirements
- **Calculates bowling scores**: The application computes the scores based on the traditional rules of bowling.
- **Manual score input**: Users can manually add scores to test the scoring system.

- **Variable number of players with names**: Users can add a variable number of players, each with their own name.
- **New game with same players**: Users can start a new game with the same players without having to re-enter their names.



## Installation

To install and run the application locally, follow these steps:


1. **Install dependencies**:
    ```
    npm install
    ```

2. **Run the application**:
    ```
    npm start
    ```

3. **Open the application**:
   Open your browser and navigate to `http://localhost:3000`.
